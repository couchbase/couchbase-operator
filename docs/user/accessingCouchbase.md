# Accessing Couchbase

After deploying Couchbase on Kubernetes or OpenShift, you can use the Kubernetes or OpenShift console or CLI to verify that the pods are all deployed and running. However, to develop, administer, or configure Couchbase, you need to access it using one of the following:
 * [the Couchbase Web Console](adminConsoleAccess.md)
 * [the Couchbase CLI](couchbaseCliGuide.md)
 * [The Couchbase REST API]()
