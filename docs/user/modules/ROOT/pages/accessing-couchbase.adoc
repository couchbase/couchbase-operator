= Accessing Couchbase

After deploying Couchbase Server on Kubernetes or OpenShift, you can use the Kubernetes or OpenShift console or CLI to verify that the pods are all deployed and running. However, to develop, administer, or configure Couchbase, you need to access it using one of the following:

* xref:admin-console-access.adoc[Couchbase Server Web Console]
* xref:couchbase-cli-guide.adoc[Couchbase CLI]
